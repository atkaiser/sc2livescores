About

As mentioned on the home page, this site aims to provide live scoring for SC2 games. It does that by trying to read the state of the game from the stream. Currently it does this by taking a screen shot every few seconds and reading all the availble information, such as names, score, supply, etc. However, for streams that are below 1080p it can be extremely hard to read the information correctly, thus the low number of supported streams. Still I will try to develop more configs for the big touraments. If you have any questions or comments, please send them to sc2livescores@gmail.com
